# Quarduped-Robot
This small project was created for a university programm. It´s goal is to get used to an array of servos and controlling them in order

## Features
- C++
- Arduino-based control
- 8x Mirco Servo 9g
- Custom soldert PCB
- Custom 3D printed mechanical frame

## Finished
- [x] wave
- [x] stand_up
- [x] reset

## Problmes
- Limited Movment
- Stability while moving due to only having 4 legs
- Limited torque
- Missing steps due to inconsistent and underpowered powersupply
